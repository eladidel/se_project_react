# WTWR (What to Wear?)

**WTWR ("What to Wear?")** is a React application that helps users choose what to wear based on the current weather. It fetches live weather data and displays suitable clothing items from a user's wardrobe.

👉 **[Live Demo](https://eladidel.github.io/se_project_react/)**

---

## Features

- **Live Weather Updates:** Displays current weather conditions and temperature.
- **Smart Clothing Suggestions:** Filters clothing based on the weather (hot, warm, or cold).
- **Add & Manage Clothing:** Modal forms to add new clothing items with image URLs and tags.
- **Temperature Unit Toggle:** Switch between Fahrenheit (°F) and Celsius (°C).
